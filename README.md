# Bunker-Game
My idea based on Worms 2

This is the only project I have ever wanted to make.

It's based on Worms 2 (game), and it's terrain editor. I want to make a setup that I tried with Worms 2 for a while when I was a kid, of making two opposing bunkers face eachother. And shoot it out. And a trench, in front of the bunker, where some of the worms can hide.

And a big cannon on top of the bunker. What would be the main gun.
